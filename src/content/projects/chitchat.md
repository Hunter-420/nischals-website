## Features

1. Google Authentication (NextAuth)
2. Create Unique Usernames
3. Send & Receive Messages in Real-time
4. Create Conversations with other users
5. Notifications (pop-ups) for new messages
6. Dot indication for which message is opened or which not
7. Users can both leave conversation as well as delete
8. Able to edit user Participants in a particular conversation either add or remove users

## Building Process

This project was built using the following technologies:

- Next.js
- NextAuth
- Prisma
- MongoDB
- Chakra UI
- Apollo GraphQL
- Node.js

The full source code is available on [GitHub](https://github.com/hunter420/ChitChat).
